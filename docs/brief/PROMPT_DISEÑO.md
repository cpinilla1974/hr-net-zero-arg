# Prompt para Diseño de Identidad Visual

## El Proyecto
Sitio web para la **Hoja de Ruta Net Zero Argentina 2050** de la industria del cemento y hormigón. Es un dashboard interactivo que muestra el progreso de descarbonización hacia 2050, con calculadora de emisiones y simulador de escenarios.

## Páginas
1. **Home**: Hero + 4 KPIs + gráfico de trayectoria
2. **Dashboard**: Indicadores en tiempo real, gráficos de progreso
3. **Calculadora**: Sliders interactivos para simular escenarios
4. **Hoja de Ruta**: Los 11 ejes de descarbonización
5. **Datos**: Descargas y metodología
6. **Sobre**: Institucional

## Valores a transmitir
- Compromiso ambiental + rigor técnico
- Transición industrial (del gris al verde)
- Confianza y transparencia
- Innovación con solidez

## Concepto visual
"Transición Verde" - Evolución del gris cemento hacia el verde sostenible, representando el camino a Net Zero.

## Necesito
1. Paleta de colores (5-6 colores)
2. Tipografías (títulos + cuerpo + datos)
3. Estilo de gráficos/visualizaciones
4. Diseño de componentes: gauges, cards de KPI, barras de progreso
5. Mockup de página principal

## Datos clave para visualizar
- Emisiones: 507 → 500 (2030) → 0 (2050) kgCO₂/tcem
- Factor clínker: 67% → 61%
- 11 ejes de descarbonización con % de aporte

## Referencias
- Las imágenes adjuntas son del documento oficial
- Inspiración: gcca.earth, climateactiontracker.org

## Contexto
Actores: AFCP (Asociación de Fabricantes de Cemento Portland), empresas (Holcim, Loma Negra, etc.), ONUDI. Argentina liderando en Latinoamérica con emisiones 18% menores al promedio global.
